# Cedar-OS
A Lite, Free, Open Source Operating System, made from scratch.

about this OS: This OS is in prereleases, it will improve over time.

# how to compile ?

This is writen in asm x86 and need to be compiled speceficly with **NASM**, by typing:
`nasm -f bin boot.asm -o boot.bin`

# how to run it?

normally you can use **qemu** to execute on a virtual machine, you could if you know scripting, code a script to turn to a img or iso and load it in virtualbox, or vmware or even real hardware, but that not recommended.
